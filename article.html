Overview of Data Visualisation

Umtha-Unathi Mavuso August, 2023 450

Data visualization has evolved from hand drawing to photo etching, often being associated with the field of computer science and animation (Li, 2020). The development of computer software has enhanced the principles of traditional data visualization by making the information easier to communicate and comprehend because the information is more cognitively appealing than traditional text-based tables and charts.

In my experience/capacity as a game designer and web developer human visual perception plays a large role in my data visualization design process in the following ways: 1. Provide displayed status information about players or characters, such as: health, score, and experience points. 2. Understanding player behavior: As a game developer I can make use of data visualization to track player behavior, such as in-game player engagement/development, how long players spend on each level, what actions they take, and what decisions they make. 3. Create more immersive experiences: Game developers make use of data visualizations to create immersive user experiences for players. 4. Data visualizations can function as narrative devices in storytelling: The use of data visualization affords the developer opportunities to develop stories and narratives within games.

In relation to me as a Web Developer: 1. User controlled/adjusted customization capabilities to various user-defined exploration scenarios and preferences is a function of web development with regards to information displayed based on data (Li, 2020). 2. The above-mentioned audience engagement model can be used as a medium to adjust displayed data in a more engaging and informative way: Therefore, we can make use of data visualization to present data in a more engaging and informative way than traditional text-based tables and charts thus increasing user engagement. 3. Improving the user experience: Data visualization can be used to create interactive dashboards/landing pages that allow our users to analyze data. 4. Tell stories with data: Data visualization can be used to tell stories with data. For example, I may decide to create infographics in order to explain complex data concepts in a simple and easy-to-comprehend manner thus it can be used to present data in a more engaging and informative way, furthermore, to improve the user experience of websites and web applications (Wallner and Kriglstein, 2015).
