Overview of Data Visualisation

Umtha-Unathi Mavuso August, 2023 1, 500

Blog Post 1 - Close Reading - Overview of Data Visualisation:

Data visualization has evolved from hand drawing to photo etching, often being associated with the field of computer science and animation (Li, 2020). The development of computer software has enhanced the principles of traditional data visualization by making the information easier to communicate and comprehend because the information is more cognitively appealing than traditional text-based tables and charts. The text provides a comprehensive overview of data visualization, including its definitions, purposes, and applications. Including the definition of data as raw facts, while information is data that has been processed and interpreted (Li, 2020). The forms of data representation include, however, are not limited to; trees, scatter plots, charts, tables, maps, diagrams, graphs, waveforms, simulations, and volume (Li, 2020) but also include computer science-dependent design systems/professions such as animation, game design, web development, graphic design. The manner in which these types of data are visually represented to users affects user engagement, user experience, and comprehension of the data being represented. This data is raw, unprocessed, and has no meaning until it is processed and organized into a wide assortment of forms, consisting of numbers, letters, symbols, and files i.e., data visualization is based upon the identification of patterns and trends in data that would be difficult to see otherwise then transformed from raw data into communicable and understandable information (Li, 2020). The integration of computer science, i.e., programming and animation, has resulted in increased attention to aesthetics and the incorporation of aesthetically pleasing design in data visualization (Li, 2020). The increased emphasis on aesthetically pleasing design is a direct result of aesthetics having been identified as being associated with improved user experience and engagement thus further entrenching the relationship between aesthetics and usability in the design of human-computer interaction referred to as HCI in the reading (Li, 2020). I believe that the main principle of data visualization is the process of transforming data into a visual representation that can be easily understood by players, users, and web visitors. To achieve this data is divided into primary and secondary data (Li, 2020) where primary data is collected directly from the source, while secondary data is collected from existing sources, and further sub-divided into characteristics of traditional data visualization of “readability, recognizability, and meaning” (Li, 2020). Readability refers to the ability of the visual representation to be understood easily, recognizability refers to the ability of the visual representation to allow users to identify previous knowledge, and meaning refers to the ability of the visual representation to convey information that people can understand. Human visual perception is the process of organizing, interpreting, and consciously experiencing sensory data, in this case, it’s visual stimuli (Li, 2020). This creates an extremely perplexing and subjective process that consists of factors such as the size, location, orientation, color, density, and texture of the elements of the visualization regarding how to manipulate those factors to engage principles of “human visual perception” to the data processed and being displayed as information (Li, 2020).

In my experience/capacity as a game designer and web developer human visual perception plays a large role in my data visualization design process in the following ways: 1. Provide displayed status information about players or characters, such as: health, score, and experience points. This is a fundamental game design mechanic because displayed status information can assist players in developing better judgment and in-game decision making to improve their performance. i.e., Implementation of player friendly sensory experiences such as the screen turning red when health is low, screen brightness gradually deteriorating as the oxygen level underwater reduces, progress markers, etc. 2. Understanding player behavior: As a game developer I can make use of data visualization to track player behavior, such as in-game player engagement/development, how long players spend on each level, what actions they take, and what decisions they make. This information is beneficial for players as it is used as a means of product quality control where the immediate identification and quickfire resolution of an issue is made possible thus affording the improvement of the game and make the game more engaging for players furthermore, as a game designer this will allow me to evaluate my game and adjust things such as difficulty settings, increased or decreased time, etc. (Li, 2020). 3. Create more immersive experiences: Game developers make use of data visualizations to create immersive user experiences for players. For example, data visualization can be used to create computer-generated visualizations of the game world/map that will help players to understand the game's mechanics ultimately, aiding player development and promoting enjoyment. 4. Data visualizations can function as narrative devices in storytelling: The use of data visualization affords the developer opportunities to develop stories and narratives within games. For example, data visualization can be used to create visualizations that show the history of the game world or to create visualizations that help players to understand the game's characters and plot.

In relation to me as a Web Developer: 1. User controlled/adjusted customization capabilities to various user-defined exploration scenarios and preferences is a function of web development with regards to information displayed based on data (Li, 2020). Application of this principle is essential for web development. I have defined web development based on data visualization as follows – web development that makes use of easily communicable and understandable UI/UX resources or as Karlsson explains visualization as a means of making data easier to understand using user-friendly sensory experience such as the amount of time spent in certain sections of each webpage expressed as a percentage of pressed, the number of visits to particular webpages/sub-pages, incorporating vanishing animations when a new webpage is being loaded, etc. (Karlsson, 2016). 2. The above-mentioned audience engagement model can be used as a medium to adjust displayed data in a more engaging and informative way: Therefore, we can make use of data visualization to present data in a more engaging and informative way than traditional text-based tables and charts thus increasing user engagement. So that whenever a user reads or looks at the data being represented on our sites or web applications, in any form, that data is understandable. Therefore, users and web visitors will be able to use the information effectively through gaining insight and knowledge (This can be done by highlighting the important data and including a short description of what data is being represented). This will make it seamless for us as web developers to communicate with stakeholders such as clients, investors, and users. Thus, ensuring the integrity of the project because everyone is on the same page. 3. Improving the user experience: Data visualization can be used to create interactive dashboards/landing pages that allow our users to analyze data. As a web developer I am also a marketer of my website/application meaning the use of marketing materials will significantly aid in promoting my website thus attracting new users and customers. Due to the development of “visualization software or applications” with the purpose to explore, visualize and analyze data I can also use data visualization analysis as a measure of performance i.e., It can be used to identify patterns and trends (Li, 2020). This information can be used to make better decisions about the design and development of the website. 4. Tell stories with data: Data visualization can be used to tell stories with data. For example, I may decide to create infographics in order to explain complex data concepts in a simple and easy-to-comprehend manner thus it can be used to present data in a more engaging and informative way, furthermore, to improve the user experience of websites and web applications (Wallner and Kriglstein, 2015). Data visualization is a powerful tool that can be used to improve the user experience of web surfers, gamers, and web/game developers. It can be used to present data in a more engaging and informative way, to improve the user experience of websites and web applications, to tell stories with data, and to analyze data. Data visualization is a valuable tool that can be used to make websites and games more user-friendly, informative, and engaging.
